# 📊 Monthly Expense Tracker

A native Android app to track your daily, weekly, and monthly expenses — built with Kotlin and Jetpack Compose.

Log expenses instantly, edit any past or future date, view smart stats grouped by expense type, and export clean PDF reports — all stored locally on your device, no internet or backend required.

📱 **[Download the APK](https://drive.google.com/file/d/1YBjXD-YHMncYbXrQ8Lyt2bcqgWKIBazx/view?usp=sharing)**

---

## ✨ Features

- **Today Expense tab** — auto-detects today's date, add multiple expenses with smart autocomplete (type "Milk" once, and it's remembered going forward)
- **Edit Expense tab** — full calendar view to add, edit, or delete expenses on any past, present, or future date
- **Expense Stats tab** — dashboard with today's, this week's, and this month's totals, plus charts grouped by expense type
- **Monthly reset** — current month's dashboard resets automatically each month, while full history stays browsable anytime via the calendar
- **PDF export** — download a clean report for today, this week, or this month with one tap
- **₹ INR support** — Indian currency formatting throughout
- **Light & dark mode**
- **Built-in support option** — raise a query directly via email from within the app

---

## 🛠️ Built With

- **Kotlin**
- **Jetpack Compose** — modern declarative UI toolkit
- **Room Database** — local offline storage
- **Navigation Compose** — bottom nav + drawer navigation
- Built and iterated using **Google AI Studio** and **Android Studio**

## 📥 Installation

### Option 1: Install the APK directly
1. Download the APK from the link above.
2. On your Android phone, open the downloaded file.
3. If prompted, allow "Install unknown apps" for whichever app you used to open it.
4. Tap Install.

### Option 2: Build from source
1. Clone this repository:
   ```
   git clone https://github.com/vijayakumar-14/Expense-Tracker.git
   ```
2. Open the project in **Android Studio**.
3. Let Gradle sync complete.
4. Connect an Android device or start an emulator.
5. Click **Run**.
